homework5.* are files pertaining to my HW5 broadcast function and 
analysis, while homework5_mpibcast.* are files using the builtin
MPI_Bcast function for comparison

To compile and generate the clog2 file for my tree broadcast function,
compile with 

    make

To compile and generate the clog2 file for the MPI_Bcast for comparison
program, compile with

    make -f Makefile.mpibcast
