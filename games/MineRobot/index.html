Mine, Robot!

My submission for the RE:Start! PSP 2022 Game Jam. You play as a robot disarming mines, with a mine sensor that keeps glitching in and out.

Downloadable from the itch.io page or Bitbucket.

Programmed in C++ using PSPDEV/PSPSDK and my own HopsLib:

  • https://pspdev.github.io
  • https://bitbucket.org/williamblair/hopslib/src/master/

Source code available at https://bitbucket.org/williamblair/restartpsp2022/src/master/
