Wii Homebrew Installation

Bought a Wii that was for sale at the local Salvation Army, so of course the first thing I wanted to do was hack it!

First, whoever donated it didn't bother removing their accounts or data from it. Also, annoyingly, it had parental controls enabled with an unknown pin... This was preventing me from factory resetting the system and even connecting to the internet.

I was able to get around this following this video:

  1. Go to the Wii settings
  2. Ensure date, time, country are up to date (change if necessary)
  3. Parental Controls
  4. "I forgot", "I forgot"
  5. Write down the confirmation number
  6. On your pc, go to https://wii.marcan.st/parental
  7. Enter the confirmation number from 2 steps previous
  8. Enter your current date, click "get reset code"
  9. On the wii, enter the code generated on the website. The wii should now have parental controls removed!

Now I could go about connecting to the internet (needed for the homebrew install). I manually entered my wireless network information as it wasn't showing up in the wii search. Figured out the security setting was wpa2 psk, based on looking on the connection settings on my phone. Also did a factory reset via "Format Wii System Memory."

For the actual homebrew channel installation, followed this video:

  1. Ensure the Wii system menu version is 4.3 (if not, do a system update)
  2. Note the letter after 4.3 (in my case U)
  3. Write down/save the system's MAC address by going to system settings
  4. On a pc, format a SD card to FAT32
  5. Go to https://please.hackmii.com/, select the system menu version, enter the MAC address, check I'm not a robot, and click "cut the [red|blue] wire" (doesn't matter which one)
  6. Extract the downloaded zip contents to the root of the SD card
  7. Make a folder called "apps" on the SD card root
  8. Go to https://wii.guide/hbb.html and click/download "Homebrew Browser." Extract the zip contents to the "app" directory on the SD card.
  9. Insert the SD card into the Wii
  10. Browse the mail (wii message board) screen from the bottom right of the Wii's home page
  11. Scroll to the previous day's date; should see a "letter bomb" logo; select it
  12. Wait a minute then press 1 to continue when it says to
  13. Make sure the homebrew channel says "Can be installed"; select continue
  14. Move to and select "Install the Homebrew Channel"
  15. Move to and select "Exit"
  16. And wahlah, the homebrew channel is installed!
