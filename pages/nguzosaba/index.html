Nguzo-Saba Typer

My submission for PDub's Holiday Game Jam 2021

My first game jam! Made over the course of about 3-4 days. Programmed in C++ using SDL2/SDL_image/SDL_mixer. Source code's available on github and bitbucket.

It's a typing game where you enter the symbol names for the principles of Kwanzaa. A prebuilt 32-bit windows executable can be downloaded from github or the itch.io page.
