NPC Escort Simulator

A submission for the SNHU PDub's Break Jam August 2022. Silly short demo of "escorting" the AI player, mainly made to learn Irrlicht engine. Downloadable from the itch.io page or Github.

Programmed in C++ with Irrlicht and https://github.com/libsdl-org/SDL_mixer.

Compiled With Visual Studio 2019 32-bit for windows. Source code available at https://github.com/williamblair/PDubsBreakJamAug2022
