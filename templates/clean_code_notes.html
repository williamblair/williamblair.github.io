These are my notes from Clean Code by Robert "Uncle Bob" Martin. I mainly wrote them to get more practice with Latex, and also to try out latex2html. These html pages were generated from the following tex/pdf:

main.tex
main.pdf

One interesting thing is that latex2html put all images as located at /usr/share/latex2html/icons, which obviously doesn't work if you actually put them online somewhere. This was an opportunity to practice my sed skills; so I changed all of the locations of the images by removing the '/usr/local/...' prefix with the following command: 

sed -ibak 's/\/usr\/share\/latex2html\/icons\///g' ./*.html

Then I just copied all of the required images to the current directory. Anyways, here are the notes:
